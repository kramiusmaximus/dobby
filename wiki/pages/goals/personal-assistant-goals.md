# Personal Assistant Goals

## Goal

Make the assistant useful as a persistent personal assistant by maintaining durable context in the wiki.

## Initial Responsibilities

- Track thoughts and ideas worth preserving.
- Organize user-provided goals and projects.
- Capture reminders and distinguish notes from actual notifications.
- Track schedule context when the user provides it.
- Maintain open questions and unresolved loops.
- Keep a chronological log of meaningful wiki changes.

## Reminder Convention

If the user asks to be reminded, create an actual automation when available. Also record durable reminder context in the wiki when it matters beyond the notification itself.

Daily task-list items should not automatically become Apple Reminders. Mark clarified on 2026-05-25 that adding daily tasks to the Reminders app is overkill for now. Create Reminders only when he explicitly asks for a reminder, notification, due date, or alert.

## Success Criteria

- The assistant can answer questions from accumulated wiki context.
- The index remains useful as the first navigation surface.
- Important conversations do not disappear into chat history.
- The user can browse the wiki in Obsidian or any markdown editor.
