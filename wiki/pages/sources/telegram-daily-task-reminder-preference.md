# Telegram Daily Task Reminder Preference

## Summary

Mark clarified that daily task lists should not automatically be added to Apple Reminders. For now, turning ordinary daily tasks into reminder notifications is overkill.

## Operating Guidance

- Do not create Apple Reminders for daily task-list items unless Mark explicitly asks for a reminder, notification, due date, or alert.
- It is still appropriate to preserve durable task context in the wiki when the items are useful beyond the current day.
- When uncertain, acknowledge the task list and avoid Reminders rather than escalating everything into notifications.
